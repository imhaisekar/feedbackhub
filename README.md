# FeedbackHub

A web-based Feedback Collection System built with Node.js, Express, MongoDB, and vanilla HTML/CSS/JS.

## Features
- User registration with OTP email verification
- JWT-based secure authentication
- Feedback submission with category and star rating
- Admin dashboard with analytics and charts
- Approve/Reject feedback moderation
- Email notifications on status updates
- CSV export functionality
- Responsive design

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Tools:** VS Code, MongoDB Compass
